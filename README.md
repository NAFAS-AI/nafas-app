# 🌬️ نَفَس — Nafas

## AI-Powered Burnout Diagnostic Tool

**120 Intelligence Layers | Arabic-First | Privacy-Focused**

### Overview
نَفَس is an AI-powered burnout diagnostic tool designed for professionals worldwide. It uses 120 layers of intelligence to detect burnout patterns through natural conversation in Arabic.

### Features
- ⚡ **Quick Check** — 60-second burnout assessment
- 💙 **Free Talk** — Open conversation with AI companion
- 📊 **Deep Assessment** — Comprehensive diagnostic dialogue
- 🧠 **VAK Detection** — Identifies visual/auditory/kinesthetic learning styles from writing
- 🆘 **Crisis Detection** — Gentle support with professional referral
- 🔒 **Privacy-First** — Anonymous journey codes, no personal data stored
- 🎙️ **Voice Input** — Arabic speech recognition
- 🌍 **4 Arabic Dialects** — Gulf, Egyptian, Levantine, Moroccan

### Tech Stack
- **Frontend:** Vanilla HTML/CSS/JS (single file, 7,500+ lines)
- **AI:** Google Gemini 2.5 Flash via serverless proxy
- **Database:** Supabase (anonymous sessions)
- **Hosting:** Vercel
- **Voice:** Web Speech API
